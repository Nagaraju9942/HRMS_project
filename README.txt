HRMS Project

Folders:
- backend: Node.js + Express + SQLite server
- frontend: React app (create-react-app style) in frontend/

Instructions:
1. Backend:
   - cd backend
   - cp .env.example .env (edit if desired)
   - npm install
   - npm start

2. Frontend:
   - cd frontend
   - npm install
   - npm start

The backend will create hrms.db automatically in backend/ folder.
